# Security policy

veetest is a client-side V testing library with no network surface or runtime services.

## Reporting a vulnerability

If you believe you have found a security issue, please open a private report via [GitHub Security Advisories](https://github.com/franklinharvey/veetest/security/advisories/new) or email the repository owner through their GitHub profile.

We will acknowledge receipt and work on a fix or assessment as soon as practical.
